With AngularJS directives, you can create custom HTML elements. For example, I created a "rant" element like this:

                    
SeahawksApp.directive('rant', function () {
return {
    restrict: 'E',
    replace: true,
    template: "< div>Don't you ever talk about me! I'm the best Cornerback in the league!< /div>"
}
});

And then I inserted it into the HTML file by simply adding:


                    < rant>< /rant> (Excluding the spaces)
